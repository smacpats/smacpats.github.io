Hey, you are on an older version of MacOS, but want to get the newest MacOS 10.15 emojis? 


1. Download the updated font file from here- https://drive.google.com/uc?export=download&confirm=yLWR&id=11UbNM03Wq9WCw3uvYoi0EknYxBkHxJlR


2. Open finder and navigate to /Library/Fonts/


3. Drop the font file into that directory


That’s it!
