Tweaks hosted here:

Cocoatop for iOS 11- D0mo’s port of cocoatop process manager to 64 bit. Since he has not hosted a repo and only debs, I have mirrored its here. Compatible with 11.0-11.3.1.

Created and ported to iOS 11 by D0mo.


PikabuLite- this tweak has a possibly unreliable repo (http://35.194.196.51/repo), so I have hosted it here as a backup plan in case the official repo dies at some point. Works on iOS 10.0-11.3.1

Created by Jeffrey Chan. 


CydiaXI 0.1- very early version of cydiaxi that keeps the home cydia page the same as it is originally. Official updated version can be found at https://cydiaxi.com/repo/

I am hosting this because I like the Cydia homepage and don’t like the changed one, and figured somebody else might think alike. 

Created by ArtikusHG. Works on iOS 11.3.1.


Libertas early beta- Jailbreak detection bypass tweak. This is the first beta released by Umanghere, and it is very buggy and probably won’t work for you. 


BackgroundKill- Kill any app automatically after closing it. Options can be configured from settings. This is the first release and may have bugs and issues. Created by u/Tonyk7.


Blacklistswitcher9 (iOS 11 port)- Port of Blacklistswitcher9 to iOS 11 by u/Tonyk7.
