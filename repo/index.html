Tweaks hosted here:

EmojiPortInstaller- Most of you are probably here for this. Using some source elements of Minimal Hosts blocker, I have made an installer for the iOS 12.1 emoji file. Basically, if you install this, it will download emojiport and the files, so you won’t need to worry about them. When it installs, it makes a backup of your current emoji file and uninstalling will restore the original emoji file. You must also have http://poomsmart.github.io/repo/ added to your sources for the installer to work. Thats about all! Should work on iOS 10.0-11.4.


Spotify adblocker- block ads on Spotify through the hosts file. .deb file


Cocoatop for iOS 11- D0mo’s port of cocoatop process manager to 64 bit. Since he has not hosted a Cydia repo and only debs, I have mirrored its here. Compatible with 11.0-11.3.1.

Created and ported to iOS 11 by D0mo. .deb file
